# Code of Conduct

本项目希望维护一个直接、尊重、以问题为中心的协作环境。

## 期望行为

- 围绕事实、复现步骤、代码和用户体验讨论问题。
- 对 bug 报告和 PR review 保持具体、可执行。
- 尊重不同设备、网络环境和使用场景带来的差异。
- 在公开讨论中主动移除账号 Cookie、私有链接、签名密钥和其他敏感信息。

## 不接受的行为

- 人身攻击、骚扰、歧视性语言或持续偏离主题的争论。
- 发布侵权资源链接、账号凭据、绕过第三方站点限制的步骤或恶意利用细节。
- 在 issue、PR 或评论中泄露他人的隐私或敏感数据。

## 执行

维护者可以编辑或删除违反本准则的内容，也可以临时或永久限制相关账号参与项目讨论。安全问题请按 [SECURITY.md](SECURITY.md) 处理。
